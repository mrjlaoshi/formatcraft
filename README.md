# 🛠 Format Tools Website

一个简单实用的在线文本格式转换工具集合。

## ✨ 功能 Features

目前已支持：

- 中文繁体 ⇄ 简体 转换
- 英文大小写转换
- 中文大写数字 ⇄ 阿拉伯数字
- 中文数字 ⇄ 阿拉伯数字

未来计划：
- JSON 格式化
- Markdown 转换
- Base64 编解码
- URL 编解码
